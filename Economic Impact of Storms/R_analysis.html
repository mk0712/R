The Health and Economic Impacts of Storms in the US since 1950

SYNOPSIS

The objective of this report is to explore the NOAA Storm Database. A particular focus lies on analyzing (1) the impact of storms on health (i.e., how many fatalities and injuries are associated with what kind of storm) and (2) the imapct of storms on the US economy (i.e., how much damage measures in USD may be associated with what kind of storm). I analyzed the data from 1950 to Nov 2011.

Please note: This report is part of my assignment for the “Data Specialization / Reproducible Research” lecture offered by John Hopkins University.

The reminder is divided in two main parts. First, in part A I will explain how I processed the data and also give you the respective R code. Second, in part B I will show you the results and together with three charts that illustrate my findings.

5.2 Clean subsets

Step 5.2 transforms the exponent columns of the tables prop and crop, because the given data is not tidy yet. In particular, I first assign the exponent zero to the characters space, ?, - and +. Second, I assign the exponent 2 to H and h, the exponent 3 to k and K, and the exponent 6 to M and m. As the column is automatically read as a factor that does not contain the element 9, I cannot assign the exponent 9 in this step. Hence, I us a rather quick and dirty approach for this: I simply use an if-statement later on to filer all Bs.

for (i in 1:nrow(prop)) {

    if (as.character(prop$exp[i]) == "" | as.character(prop$exp[i]) == "?" | 
        as.character(prop$exp[i]) == "-" | as.character(prop$exp[i]) == "+") {
        prop$exp[i] <- 0
    } else if (as.character(prop$exp[i]) == "H" | as.character(prop$exp[i]) == 
        "h") {
        ## hecta
        prop$exp[i] <- 2
    } else if (as.character(prop$exp[i]) == "K" | as.character(prop$exp[i]) == 
        "k") {
        ## kilo
        prop$exp[i] <- 3
    } else if (as.character(prop$exp[i]) == "M" | as.character(prop$exp[i]) == 
        "m") {
        ## mil
        prop$exp[i] <- 6
    } else if (as.character(prop$exp[i]) == "B" | as.character(prop$exp[i]) == 
        "b") {
        ## bil --> not in factor
        prop$exp[i] <- "B"
    }

}

5.3 Compute damage per event

Step 5.3 computes fills the table totalDmgDT. This table has one row for each event. This row shows the event name, the crop damage, the property damage, and the total economic damage.

for (i in 1:length(eventVector)) {

    x <- as.character(eventVector[i])

    prop_damage <- as.numeric(0)  ## this shows damage in thousands!
    crop_damage <- as.numeric(0)  ## this shows damage in thousands!

    for (j in 1:nrow(prop)) {

        if (x == as.character(prop$event[j]) & as.character(prop$exp[j]) == 
            "B") {

            prop_damage <- prop_damage + prop$damage[j] * 1e+09

        } else if (x == as.character(prop$event[j])) {

            prop_damage <- prop_damage + prop$damage[j] * 10^as.numeric(prop$exp[j])

        }

    }

    for (j in 1:nrow(crop)) {

        if (x == as.character(crop$event[j]) & as.character(crop$exp[j]) == 
            "B") {

            crop_damage <- crop_damage + crop$damage[j] * 1e+09

        } else if (x == as.character(crop$event[j])) {

            crop_damage <- crop_damage + crop$damage[j] * 10^as.numeric(crop$exp[j])

        }

    }

    total_damage = prop_damage + crop_damage

    additionalEventList <- list(x, prop_damage, crop_damage, total_damage)
    totalDmgDT <- rbind(totalDmgDT, additionalEventList)

}
